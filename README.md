# ServiceSetu – Responsive Startup Landing Page

## Project Overview

ServiceSetu is a modern responsive startup landing page developed using HTML, CSS, and JavaScript.  
The platform is designed to connect users with trusted local service providers such as plumbers, electricians, cleaners, painters, and home maintenance professionals.

The project focuses on:
- Responsive Web Design
- Professional UI/UX
- Interactive Frontend Features
- Cross-Page Theme Management
- Modern Landing Page Development

---

# Project Objective

The objective of this project is to create a professional responsive company landing page with modern frontend features and smooth user experience.

The application demonstrates:
- Responsive layouts
- Mobile-first design
- Authentication UI
- Dynamic animations
- Dark mode implementation
- JavaScript interactivity
